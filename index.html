Thoughtful Planning

My cocktail napkin wireframes

A lot of time is wasted refactoring code that can be avoided when you plan out your structure.

I alway start a project off by sketching out the layout and then wirframing the structure. Special attention is paid to the items that will be breaking down into a single column view for mobile. These items must be structured in a way that allows the manipulation of them together on desktop, and then separate when the screen size drops to mobile.

I also love putting on my UX hat and thinking about how my content is being perceived and tweaking it to best engage my audience. These are some of the sketches for my latest projects

Schwab Trading sitelets

New Responsive Trading Pages

Third Rail Creative provided an updated design for 9 pages. This was the first time our client developed responsive.

My first project with Third Rail. They did a great job on the design! I was the lone developer to bring the PSD's to life. We focused on 4 media break points to size it down for different viewing experiences.

We were bound by tying in to the old code and frameworks. This was not a ground up project. The first drafts had fairly large file sizes and beastly stylesheets as a result. I grouped as many assets and code into global folders as I could and compressed the images to minimize load times for the user. We ended up with a total project at half the size as the first draft.

Banner ads

Google ads through DoubleClick

This was a tough project for 2 reasons.

The overall file size had to be kept under 40Kb. That meant no jquery or frameworks. I had to write all of the click events in vanilla Javascript and was limited to CSS animations. It also meant limited assets. I had to investigate the best ways to compress image assets with lossless quality.

The other reason it was tough was that I had to abandon my beloved development environment and create these in Google Web Developer. Altogether not a bad program, but it was unfamiliar. The generated files are then published to DoubleClick Studios and sent to the advertising agency.

I'm glad to have had this challenge that helped me gain the experience to develop HTML/CSS/JS content in an extremely tight way.
